Project Overview

AutoFill Resume Pro is a C# WinForms desktop application that generates professional resumes in A4 and US Letter formats. It streamlines resume building with customizable colors, icons, fonts, and autosave. The project began as a VBA prototype and grew into a full branded desktop application with installer, GitHub repository, and tutorial videos.

The Problem

  • Jobseekers struggle with resume formatting and consistency.
  • Many templates break across A4/US Letter; icons/colors often don't transfer well.
  • Non-technical users need a tool that's simple, professional, and flexible.

Market Research & User Insights

  • Competitor Scan — Reviewed how existing resume sellers structure products.
  • Keyword Research (Google + E-Rank) — Identified strong search demand; niche proven by profitable sellers under 16 months old.
  • User Feedback Review — Analysed reviews + community forums.

Key Research Insights:

  • Resume templates often break across formats (A4 vs. US Letter), misaligning margins/icons.
  • Many users lack access to or struggle editing in MS Word/Google Docs.
  • Demand is high for both A4 (NZ/AU/Europe) and US Letter (US/Canada) versions.

Opportunity Insights:

A clear gap existed for a desktop application that generates resumes, cover letters, and references seamlessly — no reliance on Word/Docs or advanced formatting knowledge.

Goal

  • Create an app that automatically generates resumes in multiple formats.
  • Ensure ease of use (one click = formatted resume).
  • Include branding & customization (color pickers, icons, fonts).
  • Deliver a professional product for portfolio/demo release.

Tools & Technologies

  • Excel VBA Prototype — Early chatbot + resume generator logic.
  • C# WinForms (Visual Studio 2022/2025) — Desktop app development.
  • Inno Setup — Branded installer with custom icons and copyright messaging.
  • GitHub — Version control + public portfolio repository.
  • Filmora 13 — Screen recording, editing, AI voiceover for tutorials.
  • Canva — App/installer icons + marketing mockups.

Process

  1. Prototype(Excel VBA) — Built chatbot + resume generator logic.
  2. User Flow Mapping — Designed step-by-step flow (input → preview → export).
  3. Development in C# — Created A4 + US Letter forms with custom icons, fonts, autosave.
  4. Testing — Verified release mode stability (labels/icons load correctly, color persistence).
  5. Branding & Installer — Designed branded app/installer icons, added installer copyright message.
  6. Tutorial Creation — Recorded YouTube video walkthrough with AI voiceover.

Challenges & Learnings

  • Debugging issue: A4 labels/icons wouldn't load in Release mode → fixed for stability.
  • Ensured color persistence across forms with fallback logic.
  • Trial+ error handling of installer branding + .ico sizes.
  • Balanced technical detail vs. usability: simple UI + flexible customization.

Outcome

  • Fully functional Windows desktop app (safe in release mode).
  • GitHub repo showcasing VBA prototype + C# build.
  • YouTube tutorials with AI voiceover.
  • Branded installer ready for distribution.

Impact & Reflection

What I achieved

  • Took project from concept → prototype → full product release.
  • Demonstrated skills across the entire lifecycle: design → build → test → package → brand → teach.
  • Built a cohesive brand identity (icons, installer, marketing assets).
  • Validated learning pathway: started with Coursera resume-writing course → VBA prototype → C# WinForms build.
  • Published a transparent product release + documentation on GitHub.

What I learned

  • Importance of iterative testing (debugging, test environments, informal family/friend feedback).
  • Value of branding + user experience consistency.
  • Adjusted color palette (too grey → warmer gold/neutrals) for a more approachable, professional feel.

Future Enhancements / Next Steps

  • Web-Based Platform — Responsive web app for broader accessibility.
  • User Accounts & Dashboard — Save progress, track applications (Procore-style interface).
  • Job Application Tracker — Monitor submissions, deadlines, outcomes.
  • Generative AI Assistant — Resume/cover letter advice, interview prep; powered by RAG pipeline (n8n + Supabase).
  • Smart Job Description Parsing — Extract keywords + tailor resumes automatically (inspired by JobHunnt).
  • Expanded Document Library — Thank-you/resignation letters, professional development tracker, swappable templates, certificate storage,(future scope) secure contracts.
  • Platform-Wide Styling Consistency — Extend the global styling engine so that any changes to fonts, colors, or formatting automatically apply across all documents (resume, cover letter, references) for a seamless user experience.
